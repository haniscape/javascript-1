JavaScript Basics - 1

JavaScript variables and data types

Primitive Data types: Number , String , Boolean , Undefined , Null and Symbol

Non-primitive/Reference Data types: Objects and Arrays

JS has Dynamic Typing: the data types are automatically assigned


Variables

  • Variable naming convention is to use the camelCase notation
  • start variable name with a letter, underscore , or dollar sign 
    var hurghada;
var _underScore;
var $dollarSign;
  • variable name cannot start with numbers, or symbols except for $ and underscore
  • variables name cannot have a symbol in the middle
  • variable name cannot be reserved KEYWORDS 
    • var _underScore;
var $dollarSign;
var if;
